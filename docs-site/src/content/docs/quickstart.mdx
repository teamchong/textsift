## Install

Two packages, same `PrivacyFilter` API. Pick one based on bundle size + auto-fallback need:

```sh
# Lean: 76 KB gzipped, native tokenizer, no transformers.js dep.
# Recommended for browser apps that want the smallest bundle.
npm install textsift-core

# Full: 228 KB gzipped, adds transformers.js fallback for runtimes
# without WebGPU and without SIMD-capable WASM. Drop-in for the
# old single-package install.
npm install textsift
```

The rest of this page uses `textsift-core` — substitute `textsift` if you went that way.

## Redact

```ts
import { PrivacyFilter } from "textsift-core";

const filter = await PrivacyFilter.create();

const { redactedText, spans } = await filter.redact(
  "Please email me at jane.doe@example.com about invoice 4511.",
);
// redactedText = "Please email me at [private_email] about invoice [account_number]."
// spans        = [ { label: "private_email", ... }, { label: "account_number", ... } ]
```

## Detect (no redaction applied)

```ts
const { spans, containsPii } = await filter.detect(
  "Call me at (555) 123-4567.",
);
// spans = [ { label: "private_phone", start: 11, end: 25, text: "(555) 123-4567", ... } ]
// containsPii = true
```

## Batch

```ts
const results = await filter.redactBatch([
  "First message.",
  "Second message with PII: jane@example.com.",
]);
```

## Pick a backend

By default `PrivacyFilter.create()` picks WebGPU when available, otherwise the WASM backend. The umbrella `textsift` additionally falls back to transformers.js on browsers without WebGPU. Force a specific path:

```ts
// Fastest: WGSL compute shaders, requires WebGPU + shader-f16.
const gpu = await PrivacyFilter.create({ backend: "webgpu" });

// Universal: Zig + WASM + SIMD128. Works everywhere. Multi-threaded
// when the page is cross-origin-isolated (COOP/COEP headers set).
const wasm = await PrivacyFilter.create({ backend: "wasm" });
```

See the [Backends](/textsift/backends/) page for the full compatibility table.

## Dispose when done

```ts
filter.dispose();
```

Releases GPU buffers and WASM memory. Safe to call multiple times.
